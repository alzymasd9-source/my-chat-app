# My Chat App

تطبيق شات بسيط مبني بـ Node.js يعمل مباشرة في المتصفح.

## المميزات
- شات مباشر بين المستخدمين
- واجهة بسيطة وسريعة
- بدون قاعدة بيانات - يعمل على الذاكرة المؤقتة

## الملفات
- `server.js` - سيرفر Node.js الرئيسي
- `app.js` - منطق التطبيق
- `index.html` - واجهة الشات
- `style.css` - تنسيق الصفحة
- `package.json` - المكتبات المطلوبة والأوامر

## طريقة التشغيل

1. نزّل المشروع:
```bash
git clone https://github.com/alzymasd9-source/my-chat-app.git
cd my-chat-app
npm install
npm start
انسخه وحطه في ملف `README.md` وارفعه على GitHub. 

تبغى أعدله لك حسب اللي يسويه الكود بالضبط؟ لو ترسل لي محتوى `server.js` أضبط الوصف عشان يطابق شغله الفعلي.
